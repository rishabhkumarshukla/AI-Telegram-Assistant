# 🤖 AI-Powered Telegram Assistant

A professional automation bot built with **n8n** that provides real-time updates and entertainment.

### ✨ Key Features
- **🗞️ Live News:** Fetches the latest global headlines.
- **😂 Daily Entertainment:** Random jokes and fun cat facts.
- **💡 Smart Advice:** On-demand life advice and inspirational quotes.
- **🧠 Advanced Logic:** Smart routing for commands and a fallback system for unknown inputs.

### 🛠️ Tech Stack
- **n8n:** Workflow automation & logic.
- **Telegram Bot API:** User interface.
- **NewsAPI:** Live data source.

### 📂 Getting Started
1. Download the `Telegram Message Handler.json` file from this repo.
2. Import it into your **n8n** instance.
3. Set up your **Header Auth** for NewsAPI and your **Telegram Bot Token**.
4. Click **Publish** to go live!
